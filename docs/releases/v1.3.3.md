# LocalChat v1.3.3

## 中文 🚀

### 更新亮点

- **新增 Windows 原生 Quick Drop Shelf 悬浮托盘**：将文件拖到设备卡片即可快速发送，支持折叠/展开、悬停高亮、贴边呼吸条自动收起，无需打开主窗口。
- **优化动画与排版**：修复拖动文件到悬浮窗图标边缘时动画抽搐的问题。
- **优化长设备名展示**：设备卡片排版支持折行（`DT_WORDBREAK`），避免长设备名称显示不全。
- **稳定设备列表**：优化同步逻辑，在设备刷新时防止设备顺序频繁闪烁。
- **右键悬浮窗隐藏**：增加折叠状态下右键点击悬浮窗快速隐藏的上下文菜单。
- **贴边隐藏设置**：设置页新增“贴边后自动隐藏”开关（默认关闭）。

### 安装包

- `LocalChat-1.3.3-windows-x64.zip`
- `SHA256SUMS.txt`
- Android 端由于本次无专属更新，安装包保持 `v1.3.2` 版本。

### 验证

- `flutter analyze`
- `flutter test`
- `flutter build windows --release`
- `SHA256SUMS.txt` 自校验

## English 💻

### Highlights

- **Added Windows Native Quick Drop Shelf**: Drag files onto a floating device card to send them instantly, supporting collapse/expand, hover highlight, and auto-hide breathing bar when docked.
- **Fixed Animation Jitter**: Fixed window jittering when dragging files near the float window edges.
- **Improved Long Name Layout**: Device names now wrap (using `DT_WORDBREAK`) and truncate gracefully with ellipsis.
- **Stabilized Device Sync**: Preserves old ordering when syncing identical device lists to prevent visual flashing.
- **Right-Click Context Menu**: Allows users to quickly hide the drop shelf by right-clicking in collapsed state.
- **Dock Auto-Hide Setting**: Added "Auto hide when docked" toggle in settings (disabled by default).
